现在的OpenMPL暂时不支持这种gds文件，因此需要把所有cell flatten，即不存指针，而是直接存指向的cell本身。

EDA terms

Posted on 2020-02-26 Edited on 2020-02-29 In study Valine:

Floorplanning

Floorplanning is typically considered the first stage of VLSI physical design.

  • Given a set of hard blocks (whose shapes cannot be changed) and/or soft blocks (whose shapes can be adjusted) and a netlist,
  • floorplanning determines the shapes of soft blocks and assembles the blocks into a rectangle (chip)
  • so a predefined cost metric (such as the chip area, wire length, wire congestion) is optimized

Placement

Placement is the process of assigning the circuit components into a chip region. (sounds similar with floorplanning?)

  • Given a set of fixed cells/macros, a netlist, and a chip outline,
  • placement assigns the predesigned cells/macros to positions on the chip
  • so that no two cells/macros overlap with each other (i.e., legalization) and some cost functions (e.g., wire length, congestion, and timing) are optimized

Routing

After placement, routing defines the precise paths for conductors that carry electrical signals on the chip layout to interconnect all pins that are electrically equivalent.

魔幻的2020

Posted on 2020-02-02 Edited on 2020-02-01 In life Valine:

刚刚呼朋唤友又刷了一个本，吃下一粒褪黑素后，潦草的扫了下新闻：一些太过魔幻的东西让我辗转反侧无法入睡，索性起床写下一些东西。

肺炎

从临近过年大概一两个星期，香港这边就是如临大敌的气氛。我和爸妈视频，感觉作为医务工作者的他们，反倒透露出一股歌舞升平的味道，我爸还嘟囔着：香港那帮年轻人不是最爱戴口罩了吗。

囧

不过事情也确实超乎了我的预料：没想过会导致全国范围内的娱乐萧条，没想过能超过SARS，没想过一天确诊数能几千几千的增加。就像是任何事件的导火索一样，这个肺炎给这个魔幻主义的2020一次完美的导火索/开幕式。

记得小学六年级的汶川大地震，那时候看着新闻，妈妈临死前把孩子抱在怀里，把自己最后对孩子的祝语写在了手机上：

亲爱的宝贝，如果你能活着，一定要记住我爱你

那时候有人把这个故事写成了一首歌。考试的时候我做完卷子，就在草稿纸上一边写着歌词一边哭，监考老师还问我怎么了，然后给我递纸。

我一直都是一个容易感动的人，我太喜欢太尊敬太崇拜这种爱了。所以这次看到一些新闻的时候，或像刀割一样难受，或像眼睛想要洗澡一般涌出一股暖流，比如援鄂医疗队里的小护士抱着爸妈哭，比如春晚上几位主持人振聋发聩的演讲，比如日本友人的援助：山川异域，风月同天。

红十字会

除了感动，也有愤怒。

我很鄙视愤恨发国难财的人。这种就像国难当头的带路党，可耻可恨。因此初看到仁爱医院/仁和医院/红十字会的新闻后，真的恨到心里咬牙，忽然又想到自己捐的那点小钱，赶紧打开手机查了下，嘘。。不是捐给红十字会。

可是除了不捐给它，我还能做什么吗？颇有点报国无门，怀帝阍而不见，奉宣室以何年的意思。

然后整个事件开始迸发：迂腐的官僚们也许也想不到互联网时代的能量。太多太多的丑闻，一件一件血淋淋的揭开在人们面前，我甚至有点明白鲁迅先生当年弃医从文了：我学计算机也救不了国人。

我不是一个学政治社会学的人，也发表不了高谈阔论，只能说说自己对于国家的希冀。

我是个比较偏向”五毛“的角色，喜欢中国历史，中国文学，也是有着如郭靖般“侠之大者”家国情怀的人，因此我也不指望、不相信走台湾数典忘祖的路子会好过现在，就像钱穆先生说的，中西文化政治的发展，是在两条发展线上各自独立的东西，中国能发展几千年朝代更替也肯定有其适合中国社会的地方。同时，我也确实才看着中国各方面在逐渐变好：经济发展、政治清明、科研提高，就像外出游玩的旅客：当年红卫兵那一辈的大爷大妈终会老去，我们这一代的大爷大妈素质肯定会越来越高。而且比如十天修医院、廉价学费、不用排长队的医疗资源，许多方面社会主义还是有可取之处的（手动狗头）。（当然我也没在资本主义国家呆过，这些对比信息也只是从互联网获取）

但是我对这个国家的政治体系真的失望了太多次。太多次的敷衍办事、贪污腐败，各种滑稽到只可能在烂分电视剧或者嘲讽的脱口秀出现的场景反复的出现在这个国家。西方的所谓民主不能让一个政策强有力的贯彻执行，但我们的专制也让一个政策变为空谈。

我真提不了什么意见：我知道的只有如何解图着色问题等诸如此类“书呆子”的标配，倘若我当初去做了记者，也许还能洋洋洒洒写个几千字。我只希望有朝一日，能有一个理想政体，强大而不强暴，以人为本的同时又有卓识远见。算了。也许是梦里才有吧。

科比

第一眼看到这个的时候，应该和许多篮球迷一样，是懵的。

然后去各个网站确认了消息。坦白来说，我不是个科密，作为一个詹密，对科比有尊重和喜爱，因此更多的还是难以置信。一个这样的仿佛去年还在看他打球的巨星，就这样离世了。

不得不感慨生命的脆弱，以前听过把每天活成生命的最后一天这样的教诲，但只有这样的事实才真正让我反思这句话。倘若这是我的最后一天，我也得像昙花一样，用全心爆发出自己的炫彩（说人话，就是不要虚度光阴）。

自己的人生

前段时间为了给爸妈“科普”我未来可能的方向，专门给他们拉了个群说了下我工作上的三种可能。

十七岁的时候不太明白选专业的意义，现在好像有了一个概念。把人生的道路比作一个从圆点出发的线，选的专业就是对应的一条狭窄的锥形区域了，可能区域外有一些你认可的“宝藏”，但是要去探寻锥形外的宝藏，实在是太难了。不过万幸的是，我这小小的锥形里面也洋溢着对我而言的宝藏。但这也只是对我而言，世界上有太多十七岁的少年，懵懂的选择了一个不喜欢的锥形，最后那条线只能挣扎着弯弯曲曲的跑向黑暗中。

Graph Isomorphsim Network Illustration

Posted on 2020-01-12 In study Valine:

Graph Isomorphism Network 从数学理论的角度证明了，在验证graph是否同构(isomorphism)上，aggregation based gcn的能力不可能超过WL test。并以此提出了一个简单的graph isomorphism network，实现了能力范围内的最大效果。

本文是对知乎大佬论文解析的补充。

WL test

  • The initial label of each node also influences the isomorphism. For example, $G_1$ and $G_2$ are graphs with same structure while $v^i_1,v^i_2$ has different labels, then the two graphs should be determined as non-isomorphism.

Lemma 2
